### 🚀 Featured Projects

#### 🌦️ [ESPMeteo](https://github.com/Morki-Tech/ESPMeteo)

An advanced, open-source weather station built around the **ESP32-S3** microcontroller. It handles sensor data acquisition, power management, and real-time environment monitoring.

- **Tech used:** C++, ESP-IDF / Arduino Framework, ESP32-S3 hardware layout.
- Check out the code and documentation here: [Morki-Tech/ESPMeteo](https://github.com/Morki-Tech/ESPMeteo)

A custom-built multi-tool for hackers and geeks, replicating and expanding upon core functionalities like Sub-GHz, RFID, and IR protocols using custom developer boards and modular architecture.
